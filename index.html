    Django PDF Export Documentation¶

    Welcome to the Django PDF Export documentation. This package provides a powerful and flexible way to export your Django model data to beautifully formatted PDF documents directly from the Django admin interface.

    Overview¶

    Django PDF Export adds PDF export capabilities to your Django admin interface with support for:

    • Multiple orientations (portrait and landscape)
    • Customizable fonts and styling
    • Configurable layouts
    • Unicode text support
    • Header and footer customization
    • Logo integration
    • And much more...

    Installation¶

    Prerequisites¶

    • Python 3.8+
    • Django 3.2+
    • ReportLab 3.6+

    Using pip¶

    pip install django-pdf-export

    From Source¶

    git clone https://github.com/yourusername/django-pdf-export.git
cd django-pdf-export
pip install -e .

    Basic Setup¶

    1. Add to INSTALLED_APPS¶

    INSTALLED_APPS = [
    ...
    'django_pdf',
]

    2. Run Migrations¶

    python manage.py migrate

    3. Set up Fonts¶

    python manage.py setup_fonts

    Usage¶

    Adding Export Actions to Your Models¶

    from django.contrib import admin
from django_pdf.actions import export_to_pdf_landscape, export_to_pdf_portrait
from .models import YourModel

@admin.register(YourModel)
class YourModelAdmin(admin.ModelAdmin):
    list_display = ('field1', 'field2', ...)
    actions = [export_to_pdf_landscape, export_to_pdf_portrait]

    Configuring Export Settings¶

    1. Navigate to Django Admin > Django PDF > Export PDF Settings
    2. Create a new configuration
    3. Set your desired options:
    4. Page Layout
    5. Font Settings
    6. Visual Settings
    7. Display Options
    8. Table Settings
    9. Mark the configuration as active

    Only one configuration can be active at a time.

    Advanced Configuration¶

    Custom Fonts¶

    The package supports TrueType fonts (.ttf). To add custom fonts:

    1. Create the fonts directory if it doesn't exist: 

      mkdir -p your_project/django_pdf/static/django_pdf/fonts/

    2. Copy your font files: 

      cp your-font.ttf your_project/django_pdf/static/django_pdf/fonts/

    3. The font will automatically appear in the font selection dropdown in PDF Export Settings

    Logo Integration¶

    1. Upload your logo through the admin interface
    2. Enable logo display in the active configuration
    3. The logo will appear in the top-right corner of exported PDFs

    Styling Customization¶

    You can customize the PDF output by extending the export actions:

    from django_pdf.actions import export_to_pdf_landscape

class CustomPDFExport(export_to_pdf_landscape):
    def get_pdf_style(self):
        style = super().get_pdf_style()
        # Customize style here
        return style

    def get_header_text(self, modeladmin):
        return "Custom Header Text"

    def get_footer_text(self):
        return "Custom Footer Text"

    Settings Reference¶

    Page Layout Settings¶

    Setting Description Default Range
    items_per_page Rows per page 10 1-50
    page_margin_mm Page margins 15mm 5-50mm

    Font Settings¶

    Setting Description Default Range
    font_name TTF font file DejaVuSans.ttf Any TTF
    header_font_size Header size 10pt 6-24pt
    body_font_size Content size 7pt 6-18pt

    Visual Settings¶

    Setting Description Default
    header_background_color Header bg color #F0F0F0
    grid_line_color Table lines color #000000
    grid_line_width Line thickness 0.25pt

    Display Options¶

    Setting Description Default
    show_header Show title True
    show_logo Show logo True
    show_export_time Show timestamp True
    show_page_numbers Show pagination True

    Troubleshooting¶

    Common Issues¶

    1. Font Not Found
    2. Ensure fonts are in the correct directory

    3. Run python manage.py setup_fonts

    4. Logo Not Displaying

    5. Check file permissions

    6. Verify logo path in settings

    7. Text Wrapping Issues

    8. Adjust max_chars_per_line setting
    9. Consider landscape orientation

    Debug Tips¶

    Enable Django's debug logging for django_pdf:

    LOGGING = {
    'handlers': {
        'console': {
            'class': 'logging.StreamHandler',
        },
    },
    'loggers': {
        'django_pdf': {
            'handlers': ['console'],
            'level': 'DEBUG',
        },
    },
}
